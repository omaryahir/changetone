# changetone

In this project you can send a txt file with the tones for example:

_myfile.txt_
```
Do      Fa     Sol
Lorem ipsum dolor sit amet
``` 

Execute the command in this way:
```
python changetone.py myfile.txt
 Main tone: Do
 Target tone: Sol 
```

Output:

_myfile.txt_
```
Sol      Do     Re
Lorem ipsum dolor sit amet
```
